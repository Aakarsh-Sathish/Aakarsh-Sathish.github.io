Self Driving Autonomous Shopping Cart for Visually Impaired

Built a shopping cart using cart chassis, batteries, and motors and used microcontrollers such as Raspberry pi and Arduino Uno

Project Overview

The main objective of the proposed system is to gain mobility and achieve navigation which has been tried to be accomplished with the help of mentioned technologies and has been successful in securing the basic functionalities required. The proposed system synergizes various components like microcontrollers, H-bridge, camera module with technologies like computer vision, machine learning and deep neural networks which is assembled altogether in the form of a cart chassis attached with a chair for the differently abled person to sit on. Microcontrollers like Arduino Uno and Raspberry Pi are used to control the cartâ€™s movements and to process the live video feed from the camera respectively. Computer vision is used to determine the optimized path to be taken by the cart using methods such as lane detection, object recognition, determining region of interest and obstacle detection. The main elements to achieve navigation are the Raspberry Pi microcontroller and the Raspberry Pi camera.

The camera is used to capture the images. The image processing and machine learning is done on the Raspberry Pi. The Raspberry Pi also acts as a master microcontroller, issuing instructions to the arduino on when to move, where to move and when to stop. Further features like Text-to-Speech, which would announce to the visually challenged person which product they have arrived at, and a shopping list which would allow the user to feed in the list of required products with the help of a shopping mart assistant allowing the cart to stop only at those products increase the usability of the cart substantially.
