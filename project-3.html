Covid 19 Trends in India

Developed a Covid-19 Dashboard in Python using datasets from Kaggle which visualized the trends Covid-19 data in India.

Link

Project Overview

The purpose of this dashboard is to spread awareness and information about Covid-19, and help visualize the various trends in both Covid-19 cases as well as vaccinations. The dashboard has various maps of different kinds which help in portraying the required information in a way for everyone to understand and appreciate it. The geographical maps represent the information across India as a whole. The users can view the data in each state, and can also compare the situation of different states using the choropleth map. The cumulative statewise graphs provide data about the cases and vaccination in every state, and are arranged in ascending order, making it easier to determine which states have the highest or lowest values for a particular variable. The rest of the graphs represent the trends in cases and vaccinations across various days. This makes it straightforward to find out the dates when the different waves of Covid-19 started, or find out the days on which there were spikes in the number of vaccines administered etc.
